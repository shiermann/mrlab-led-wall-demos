# MRLab LED Wall Demos

Interactive real-time 3D experiences for the stereoscopic LED wall in the
Mixed Reality Lab at TU Wien, built with Unreal Engine 5.

Developed as part of a Bachelor thesis at the
[VR Research Group, TU Wien](https://www.vr.tuwien.ac.at/).

## Demos

### Demo 1 – Fluid Flux Water World
A real-time water simulation powered by the
[Fluid Flux plugin](https://www.fab.com/listings/196c70cd-1283-4249-bf6b-c3019d1cbe11).
Features a MetaHuman avatar, interactive navigation via controllers,
and spatial audio through the MR audio system.

### Demo 2 – Immersive Forest Environment
A dynamic, realistic forest scene with real-time foliage, water, and a
procedural sky system. Designed for passive exploration and interactive
navigation on the LED wall.

### Demo 3 – TBD
Additional interactive 3D experience to be defined.

## Requirements

- Unreal Engine 5.7
- Fluid Flux Plugin (licensed, available within the research group)
- NVIDIA RTX 6000 (rendering server in MR Lab)
- MR Lab LED wall + controller setup

## Getting Started
```bash
git clone https://github.com/shiermann/mrlab-led-wall-demos.git
```

Open the `.uproject` file in Unreal Engine 5.7.

> **Note:** The Fluid Flux plugin is not included in this repository.
> It must be installed separately via the group's license.
